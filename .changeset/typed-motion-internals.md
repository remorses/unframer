Add `unframer/motion-internals`, an official typed entry point for deterministic renderers that need Framer Motion frame timing internals.

```ts
import {
  JSAnimation,
  MotionGlobalConfig,
  frameData,
  frameSteps,
  time,
  visualElementStore,
} from 'unframer/motion-internals'
```
